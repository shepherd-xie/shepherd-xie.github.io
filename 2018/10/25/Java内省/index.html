JavaеҶ…зңҒжңәеҲ¶

еҸ‘иЎЁдәҺ 2018-10-25

JavaеҶ…зңҒжңәеҲ¶

еҶ…зңҒ( Introspector ) жҳҜ Java иҜӯиЁҖеҜ№ JavaBean зұ»еұһжҖ§гҖҒдәӢд»¶зҡ„дёҖз§ҚзјәзңҒеӨ„зҗҶж–№жі•гҖӮ

JavaBeanжҳҜдёҖз§Қзү№ж®Ҡзҡ„зұ»пјҢдё»иҰҒз”ЁдәҺдј йҖ’ж•°жҚ®дҝЎжҒҜпјҢиҝҷз§Қзұ»дёӯзҡ„ж–№жі•дё»иҰҒз”ЁдәҺи®ҝй—®з§Ғжңүзҡ„еӯ—ж®өпјҢдё”ж–№жі•еҗҚз¬ҰеҗҲжҹҗз§Қе‘ҪеҗҚи§„еҲҷгҖӮеҰӮжһңеңЁдёӨдёӘжЁЎеқ—д№Ӣй—ҙдј йҖ’дҝЎжҒҜпјҢеҸҜд»Ҙе°ҶдҝЎжҒҜе°ҒиЈ…иҝӣ JavaBean дёӯпјҢиҝҷз§ҚеҜ№иұЎз§°дёәвҖңеҖјеҜ№иұЎвҖқ( Value Object )пјҢжҲ–вҖң VO вҖқгҖӮж–№жі•жҜ”иҫғе°‘гҖӮиҝҷдәӣдҝЎжҒҜеӮЁеӯҳеңЁзұ»зҡ„з§ҒжңүеҸҳйҮҸдёӯпјҢйҖҡиҝҮ setter/getter иҺ·еҫ—гҖӮ

Java JDK дёӯжҸҗдҫӣдәҶдёҖеҘ— API з”ЁжқҘи®ҝй—®жҹҗдёӘеұһжҖ§зҡ„ getter/setter ж–№жі•пјҢиҝҷе°ұжҳҜеҶ…зңҒгҖӮ

JDKеҶ…зңҒзұ»еә“

PropertyDescriptorзұ»

PropertyDescriptorзұ»иЎЁзӨә JavaBean зұ»йҖҡиҝҮеӯҳеӮЁеҷЁеҜјеҮәдёҖдёӘеұһжҖ§гҖӮдё»иҰҒж–№жі•пјҡ

  1. getPropertyType()пјҢиҺ·еҫ—еұһжҖ§зҡ„ Class еҜ№иұЎ;
  2. getReadMethod()пјҢиҺ·еҫ—з”ЁдәҺиҜ»еҸ–еұһжҖ§еҖјзҡ„ж–№жі•пјӣ
  3. getWriteMethod()пјҢиҺ·еҫ—з”ЁдәҺеҶҷе…ҘеұһжҖ§еҖјзҡ„ж–№жі•;
  4. hashCode()пјҢиҺ·еҸ–еҜ№иұЎзҡ„е“ҲеёҢеҖј;
  5. setReadMethod(Method readMethod)пјҢи®ҫзҪ®з”ЁдәҺиҜ»еҸ–еұһжҖ§еҖјзҡ„ж–№жі•;
  6. setWriteMethod(Method writeMethod)пјҢи®ҫзҪ®з”ЁдәҺеҶҷе…ҘеұһжҖ§еҖјзҡ„ж–№жі•гҖӮ

Introspectorзұ»

е°Ҷ JavaBean дёӯзҡ„еұһжҖ§е°ҒиЈ…иө·жқҘиҝӣиЎҢж“ҚдҪңгҖӮеңЁзЁӢеәҸжҠҠдёҖдёӘзұ»еҪ“еҒҡ JavaBean жқҘзңӢпјҢе°ұжҳҜи°ғз”Ё Introspector.getBeanInfo() ж–№жі•пјҢеҫ—еҲ°зҡ„ BeanInfo еҜ№иұЎе°ҒиЈ…дәҶжҠҠиҝҷдёӘзұ»еҪ“еҒҡ JavaBean зңӢзҡ„з»“жһңдҝЎжҒҜпјҢеҚіеұһжҖ§зҡ„дҝЎжҒҜгҖӮ

BeanUtilsе·Ҙе…·еҢ…

ApacheејҖеҸ‘дәҶдёҖеҘ—з®ҖеҚ•гҖҒжҳ“з”Ёзҡ„APIжқҘж“ҚдҪңBeanзҡ„еұһжҖ§вҖ”вҖ”BeanUtilsе·Ҙе…·еҢ…гҖӮ

BeanUtilsе·Ҙе…·еҢ…

жіЁж„Ҹпјҡеә”з”Ёзҡ„ж—¶еҖҷиҝҳйңҖиҰҒдёҖдёӘloggingеҢ…
